# dotfiles

1. Setup for using git
```
$ xcode-select --install
$ git clone https://github.com/trsxxii/dotfiles.git
```

2. Setup secret.zshrc
```
$ cd dotfiles
$ vi secret.zshrc
```

3. Exec
```
$ make all
```

4. Install Xcode with version
```
$ xcversion list
$ xcversion install XX.X
```
