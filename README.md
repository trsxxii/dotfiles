# dotfiles

Setup for using git
```
$ xcode-select --install
$ git clone https://github.com/trsxxii/dotfiles.git
```


```
$ cd dotfiles
$ sh bin/install.sh
```

Install Xcode with version.
```
$ xcversion list
$ xcversion install XX.X
```
